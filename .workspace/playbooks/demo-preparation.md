# Demo Preparation Playbook

## Goal

Prepare a stable, impressive demonstration.

---

## Verify

- Application builds
- No runtime errors
- Core features work
- Database connected
- Authentication works
- Responsive UI
- No placeholder content

---

## Demo Flow

1. Introduce problem
2. Explain solution
3. Show main features
4. Show unique features
5. Show AI features
6. Explain architecture
7. Explain future scope

---

## Finish

Restart application.

Clear browser cache.

Prepare backup screenshots.
