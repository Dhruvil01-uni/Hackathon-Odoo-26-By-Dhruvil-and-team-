# Backend Engineering Standards

## Purpose

Build secure and maintainable backend services.

---

## Folder Structure

- routes/
- controllers/
- services/
- middlewares/
- models/
- utils/

---

## API Design

- RESTful endpoints.
- Consistent responses.
- Proper status codes.

---

## Validation

Validate:

- Body
- Params
- Query

---

## Security

- Environment variables.
- Password hashing.
- Authentication.
- Authorization.

---

## Performance

- Pagination
- Query optimization
- Caching when appropriate

---

## Checklist

- Secure
- Validated
- Modular
- Error Handling
