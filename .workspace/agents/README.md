# AI Agents

This folder contains specialized AI personas.

Each agent focuses on one engineering role and should be selected based on the task.

## Available Agents

- Architect
- Frontend Engineer
- Backend Engineer
- Database Engineer
- UI Designer
- Debugger
- Code Reviewer

## Workflow

1. Read `.ai/index.md`
2. Read project context
3. Select the appropriate agent
4. Follow the related playbook
5. Use the corresponding prompt

Each agent should produce production-quality, maintainable code.
