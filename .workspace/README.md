# 🚀 Odoo-26 — AI Hackathon Workspace

A reusable AI-assisted engineering workspace designed for rapid full-stack development during hackathons.

This repository provides structured workflows, engineering standards, reusable prompts, AI agents, and project context management so AI assistants can quickly understand a project and generate high-quality code.

---

# Goals

- Rapid MVP development
- Production-quality engineering
- AI-assisted workflow
- Consistent project organization
- Reusable across future projects
