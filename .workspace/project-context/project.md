# Project Overview

## Problem Statement

TransitOps is a fleet operations ERP MVP for managing vehicles, drivers, trips, maintenance, fuel, expenses, dashboard analytics, reports, and role-based workflows.

## Goals

### Primary Goals

- Deliver a polished hackathon MVP for fleet operations.

- Implement clean full-stack architecture with React, Express, Prisma, and PostgreSQL.

- Enforce business rules for dispatch, driver eligibility, vehicle availability, maintenance, fuel, and expenses.

### Stretch Goals

- Advanced reporting and ROI analytics.

- Polished responsive ERP-style dashboard.

- Deployment on Vercel plus Render or Railway.
